Your all-in-one fitness planner: schedule workouts, track progress, visualize your journey, and stay motivated with personal stats.

## Features
- **Add** new workout plans for any date
- **View, complete, and manage** planned workouts
- **Monthly and weekly charts** to visualize progress
- **Workout streak tracker**
- **Most active workout day** summary
- **Interactive calendar** highlighting completed/scheduled workouts

## Setup Instructions

### Requirements
- [Node.js & npm](https://nodejs.org/en/download/)
- [Java (JDK)](https://adoptopenjdk.net/)
- MySQL (or compatible database)

### Backend Setup (Spring Boot)
1. Import backend code into Eclipse/IntelliJ or build with Maven/Gradle.
2. Set up your database and edit `application.properties` for database credentials if needed.
3. Start the backend app (it runs on [http://localhost:8081](http://localhost:8081)).

## How to Use
1. Add new workouts using the form.
2. Mark workouts as completed as you finish them.
3. View your progress in the dashboard, including streaks and charts.
4. Explore the calendar for your workout plan overview.
